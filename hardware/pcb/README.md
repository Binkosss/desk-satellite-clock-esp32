# PCB Files

This directory contains all PCB-related design and manufacturing files.

## Structure
- `fusion360/` – source design files (electronics + PCB)
- `gerbers/` – manufacturing outputs for board fabrication
- `exports/` – exported PDFs, images, and BOM/position files

## Workflow
1. Edit board in `fusion360/`.
2. Export production files to `gerbers/`.
3. Place documentation exports in `exports/`.
